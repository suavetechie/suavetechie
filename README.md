# 🔐 Cybersecurity Analyst Portfolio

## 👤 **About Me**

I’m a Cybersecurity Analyst with a focus on cloud security, network defense, threat detection and incident response. I bring hands‑on experience in SIEM monitoring, intrusion detection and vulnerability assessment, where I excel at analyzing log data, correlating alerts and identifying suspicious activities across hybrid infrastructures.  

Driven by curiosity and precision, I thrive on turning raw data into actionable insights while continuously mastering new tools and frameworks to stay ahead of evolving threats. My mission is to strengthen organizational resilience by combining proactive monitoring, intelligent automation and a streamlined incident response process.
